Time,Space and Complexity

Time Complexity

Time Complexity is the amount of time an algorithm takes to execute as the input size increases. It helps us measure the efficient and performance of a program An algorithm is called time efficient if it takes less execution time to solve a problem

Space complexity

Space Complexity is the total amount of memory used by an algorithm during execution. It measures how much extra space or RAM is required as the input size increases. Formula the space required depends on the number of students, which represents O(n) space complexity
